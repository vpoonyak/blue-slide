# Approved resource directory

Use this directory only when the task needs an external palette, icon, or template. Verify availability, licensing, attribution requirements, and suitability before reuse.

## Color exploration

- [HUE/360](http://hue360.herokuapp.com/)
- [Adobe Color](https://color.adobe.com/)
- [Nippon Colors](https://nipponcolors.com/)
- [Dribbble](https://dribbble.com/)

Keep the canonical BlueSlide blues unless the task requires an alternate palette. Use external palettes to find a different-hue accent or a context-specific comparison system, not to introduce arbitrary color.

## Icons

- [Flaticon](https://www.flaticon.com/)
- [The Noun Project](https://thenounproject.com/)
- [iconmonstr](https://iconmonstr.com/)
- [IcoJam](https://www.icojam.com/)
- [GraphicBurger](https://graphicburger.com/)
- [Free medical icon sets at Hongkiat](http://hongkiat.com/blog/free-medical-icon-sets/)
- [Magnific Icons](https://www.magnific.com/icons/)

Prefer simple flat icons from one coherent family. Do not mix unrelated stroke weights, perspectives, corner styles, or levels of detail.

## Templates and structural inspiration

- [Showeet](https://www.showeet.com/)
- [HiSlide](https://hislide.io/)
- [PresentationGO](https://www.presentationgo.com/)
- [Adioma](https://adioma.com/)
- [BOXIL](https://boxil.jp/)
- [Free PowerPoint Templates Design](https://www.free-powerpoint-templates-design.com/)

Use templates to study structure and composition. Rebuild the layout in the BlueSlide system; do not import a template's decorative language by default.
