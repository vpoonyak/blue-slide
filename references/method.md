# BlueSlide method

## Conceptual references

BlueSlide uses only these three presentation influences:

1. **Nancy Duarte — narrative.** Treat the audience as the hero, organize the deck around one Big Idea, create tension between “what is” and “what could be,” and end with a clear transformation and action. See [Duarte's Presentation Sparkline method](https://www.duarte.com/blog/move-presentation-audience-with-story-techniques-in-presentations/).
2. **BetterPitch (*พูดด้วยภาพ*) — investor logic.** Build decision-oriented structure with credible metrics, explicit assumptions, risks, track record, and a precise ask. BetterPitch's former domain currently redirects to Collateral Partners; use its [story, proof, and ask pitch-deck methodology](https://collateral.com/blog/what-is-a-pitch-deck) as the investor-communication layer.
3. **Chang-Yang Lin — visual clarity.** Remove decoration, expose the essence, use simple flat icons as a universal visual language, and compress information before designing. See his [flat-design principles](https://www.accupass.com/event/1703161453482262327130) and [content-distillation method](https://www.bfa.com.tw/blog/bfa-event-56-featured-lin).

## Working blend

- Use Duarte to shape the audience journey.
- Use BetterPitch to make claims decision-ready and credible.
- Use Chang-Yang Lin to make each message visually immediate.
- Combine them in that order: narrative, proof, then visual compression.

## Slide-level editing rule

Write the message before choosing the layout. Delete until one thought remains dominant. Then choose the lightest visual form that lets the audience understand that thought quickly.
